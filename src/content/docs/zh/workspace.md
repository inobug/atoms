# 使用工作区

工作区是你与 AI Agent 交互并查看生成代码的地方。

## 布局

工作区有三个面板：

### 对话面板（左侧）

向 AI Agent 发送消息。你会看到来自不同 Agent 的回复：

- 🧠 **规划师** —— 架构和设计决策
- 💻 **编码师** —— 生成的代码及说明
- 🔍 **审查师** —— 代码审查和改进建议

### 预览面板（中间）

在沙箱 iframe 中实时预览生成的应用。

功能：
- **设备模式**：在桌面、平板和手机视图间切换
- **刷新**：重新加载预览
- **全屏**：展开预览

### 文件面板（右侧）

按版本查看所有生成的文件。

## AI 模型

你可以在项目设置中切换 AI 模型：

- **Claude**（Anthropic）—— 默认选项，适合复杂应用
- **GPT-4**（OpenAI）—— 备选方案

## 获得更好结果的建议

1. **具体描述** —— 包含布局、颜色和功能的细节
2. **逐步迭代** —— 从简单开始，逐步增加复杂度
3. **使用模板** —— 用模板快速开始常见应用
4. **参考示例** —— 提及你喜欢的网站或风格

> **注意**：生成的代码在沙箱 iframe 中运行，使用 CDN 加载的库。
