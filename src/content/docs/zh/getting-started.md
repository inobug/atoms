# 快速开始

欢迎使用 Atoms —— 一个由多 Agent 协作驱动的 AI 代码生成平台。

## 什么是 Atoms？

Atoms 使用一组 AI Agent 来帮助你构建 Web 应用：

- **规划师 Agent** 🧠 —— 分析需求，设计架构
- **编码师 Agent** 💻 —— 生成完整可运行的代码
- **审查师 Agent** 🔍 —— 审查代码质量，提出改进建议

## 快速上手

### 1. 创建账户

访问登录页面，使用邮箱和密码创建新账户。

### 2. 创建项目

从仪表板，点击「新建项目」或在 AI 输入框中描述你想构建的内容。

### 3. 描述你的应用

告诉 AI 你想构建什么。尽量具体地说明功能、设计和交互。例如：

```
构建一个现代化的着陆页，包含 Hero 区域、
功能网格、推荐轮播和行动号召底部。
使用简洁专业的设计风格。
```

### 4. 观看 AI 构建

多 Agent 系统将会：
1. 规划架构方案
2. 生成完整代码
3. 审查并优化

### 5. 预览并迭代

在工作区查看实时预览，继续对话来优化你的应用。

## 支持的模板

| 模板 | 描述 |
|------|------|
| 着陆页 | Hero、功能介绍、推荐语、CTA |
| 待办应用 | 带筛选的任务管理 |
| 数据面板 | 图表和表格的分析面板 |
| 产品页 | 带画廊和购物车的电商页面 |

> **提示**：你可以使用模板作为起点，然后继续自定义修改。
