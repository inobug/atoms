# Getting Started

Welcome to Atoms — an AI-powered code generation platform driven by multi-agent collaboration.

## What is Atoms?

Atoms uses a team of AI agents to help you build web applications:

- **Planner Agent** 🧠 — Analyzes your requirements and designs the architecture
- **Coder Agent** 💻 — Generates complete, runnable code
- **Reviewer Agent** 🔍 — Reviews code quality and suggests improvements

## Quick Start

### 1. Create an Account

Visit the login page and create a new account with your email and password.

### 2. Create a Project

From the dashboard, click "New Project" or describe what you want to build in the AI input box.

### 3. Describe Your App

Tell the AI what you want to build. Be specific about features, design, and functionality. For example:

```
Build a modern landing page with a hero section, 
features grid, testimonials carousel, and a 
call-to-action footer. Use a clean, professional design.
```

### 4. Watch AI Build

The multi-agent system will:
1. Plan the architecture
2. Generate the code
3. Review and optimize

### 5. Preview and Iterate

See the live preview in the workspace, and continue chatting to refine your app.
