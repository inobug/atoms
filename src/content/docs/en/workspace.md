# Using the Workspace

The workspace is where you interact with AI agents and see your generated code.

## Layout

The workspace has three panels:

### Chat Panel (Left)

Send messages to the AI agents. You'll see responses from:

- 🧠 **Planner** — Architecture and design decisions
- 💻 **Coder** — Generated code with explanations
- 🔍 **Reviewer** — Code review and improvements

### Preview Panel (Center)

Live preview of your generated application in a sandboxed iframe.

Features:
- **Device modes**: Switch between desktop, tablet, and mobile views
- **Refresh**: Reload the preview
- **Fullscreen**: Expand the preview

### Files Panel (Right)

View all generated files organized by version.

## AI Models

You can switch between AI models in project settings:

- **Claude** (Anthropic) — Default, great for complex applications
- **GPT-4** (OpenAI) — Alternative option

## Tips for Better Results

1. **Be specific** — Include details about layout, colors, and features
2. **Iterate** — Start simple and add complexity gradually  
3. **Use templates** — Start from a template for common app types
4. **Reference examples** — Mention specific websites or styles you like

> **Note**: Generated code runs in a sandboxed iframe and uses CDN-loaded libraries.
